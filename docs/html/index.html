SDL2Gui Documentation

Introduction

SDL2Gui is a weirdly complex graphical user interface library designed for low performance and the API that will cause the developer to slam their desk. This documentation provides an overview of the API, including classes, functions, and modules.

Features

  • Hard-to-use widget system
  • Has basic GUI components and an event system
  • Cross-platform support
  • Horribly unoptimized

Getting Started

To use this, include the main header in your project:

#include <lfmain.h>
lfmain.h

The whole library is included in the namespace GUILib.

You can use features of this library directly from now on! If there's nothing interfering with the functions, you can use

using namespace GUILib;
